# CZ2006-Backend
To develop an online appointment making system for dental clinics which consists of end user mobile applications and an admin portal for clinics.
This is the repository for backend 

## Tech Stack
### Language
- Java 
### Frameworks
- Spring 
- Spring MVC
### Others
- Apache-tomcat-9
- Mysql 
- Intellij IDEA 2020.3
